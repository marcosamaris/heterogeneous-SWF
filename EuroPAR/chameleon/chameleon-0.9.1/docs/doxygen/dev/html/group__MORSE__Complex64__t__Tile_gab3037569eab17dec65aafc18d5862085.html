int MORSE_zgetrf_nopiv_Tile ( MORSE_desc_t *  A)

MORSE_zgetrf_nopiv_Tile - Computes the tile LU factorization of a matrix. Tile equivalent of MORSE_zgetrf_nopiv(). Operates on matrices stored by tiles. All matrices are passed through descriptors. All dimensions are taken from the descriptors.

Parameters
[in,out]AOn entry, the M-by-N matrix to be factored. On exit, the tile factors L and U from the factorization.
Returns
Return values
MORSE_SUCCESSsuccessful exit
>0if i, U(i,i) is exactly zero. The factorization has been completed, but the factor U is exactly singular, and division by zero will occur if it is used to solve a system of equations.
