MORSE_Desc_Create_User - Create matrix descriptor for general applications.

Parameters
[out]descOn exit, descriptor of the matrix.
[in]matMemory location of the matrix. If mat is NULL, the space to store the data is automatically allocated by the call to the function.
[in]dtypData type of the matrix:
  • MorseRealFloat: single precision real (S),
  • MorseRealDouble: double precision real (D),
  • MorseComplexFloat: single precision complex (C),
  • MorseComplexDouble: double precision complex (Z).
[in]nbNumber of rows and columns in a tile.
[in]mNumber of rows of the entire matrix.
[in]nNumber of columns of the entire matrix.
[in]p2d-block cyclic partitioning, number of tiles in rows.
[in]q2d-block cyclic partitioning, number of tiles in columns.
[in](*get_blkaddr)(const MORSE_desc_t *A, int m, int n) A function which return the address of the data corresponding to the tile A(m,n).
[in](<em>get_blkldd)(const MORSE_desc_t *A, int m) A function that return the leading dimension of the tile A(m,).
[in](*get_rankof)(const MORSE_desc_t *A, int m, int n) A function that return the MPI rank of the tile A(m,n).
Returns
Return values
MORSE_SUCCESSsuccessful exit
