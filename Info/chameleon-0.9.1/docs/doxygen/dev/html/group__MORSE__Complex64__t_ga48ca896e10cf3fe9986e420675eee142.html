MORSE_zunmlq - Overwrites the general complex M-by-N matrix C with 

            SIDE = 'L'     SIDE = 'R'

TRANS = 'N': Q * C C * Q TRANS = 'C': Q**H * C C * Q**H

where Q is a complex unitary matrix defined as the product of k elementary reflectors 

  Q = H(1) H(2) . . . H(k)

as returned by MORSE_zgeqrf. Q is of order M if SIDE = MorseLeft and of order N if SIDE = MorseRight.

Parameters
[in]sideIntended usage: = MorseLeft: apply Q or Q**H from the left; = MorseRight: apply Q or Q**H from the right.
[in]transIntended usage: = MorseNoTrans: no transpose, apply Q; = MorseConjTrans: conjugate transpose, apply Q**H.
[in]MThe number of rows of the matrix C. M >= 0.
[in]NThe number of columns of the matrix C. N >= 0.
[in]KThe number of rows of elementary tile reflectors whose product defines the matrix Q. If side == MorseLeft, M >= K >= 0. If side == MorseRight, N >= K >= 0.
[in]ADetails of the LQ factorization of the original matrix A as returned by MORSE_zgelqf.
[in]LDAThe leading dimension of the array A. LDA >= max(1,K).
[in]descTAuxiliary factorization data, computed by MORSE_zgelqf.
[in,out]COn entry, the M-by-N matrix C. On exit, C is overwritten by Q*C or Q**H*C.
[in]LDCThe leading dimension of the array C. LDC >= max(1,M).
Returns
Return values
MORSE_SUCCESSsuccessful exit
<0if -i, the i-th argument had an illegal value
