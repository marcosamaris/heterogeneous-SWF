MORSE_zungqr_Tile - Generates an M-by-N matrix Q with orthonormal columns, which is defined as the first N columns of a product of the elementary reflectors returned by MORSE_zgeqrf. All matrices are passed through descriptors. All dimensions are taken from the descriptors.

Parameters
[in]ADetails of the QR factorization of the original matrix A as returned by MORSE_zgeqrf.
[in]TAuxiliary factorization data, computed by MORSE_zgeqrf.
[out]QOn exit, the M-by-N matrix Q.
Returns
Return values
MORSE_SUCCESSsuccessful exit
