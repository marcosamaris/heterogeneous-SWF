MORSE_zgetrs_nopiv_Tile - Solves a system of linear equations using previously computed LU factorization. Tile equivalent of MORSE_zgetrs_nopiv(). Operates on matrices stored by tiles. All matrices are passed through descriptors. All dimensions are taken from the descriptors.

Parameters
[in]AThe tile factors L and U from the factorization, computed by MORSE_zgetrf_nopiv.
[in,out]BOn entry, the N-by-NRHS matrix of right hand side matrix B. On exit, the solution matrix X.
Returns
Return values
MORSE_SUCCESSsuccessful exit
